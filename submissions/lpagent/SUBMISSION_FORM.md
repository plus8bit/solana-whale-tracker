## Project Description

```txt
LP Whale Signal Agent extends my Solana Whale Tracker with LPAgent.io liquidity intelligence.

It uses LPAgent pool discovery to rank Meteora pools by fee opportunity, volume, TVL, organic score, and risk. For the highest-ranked pool, the app includes a Zap-In Transaction Planner using LPAgent's Zap-In workflow, then identifies top LP wallets that can be monitored with Solana Whale Tracker.

The goal is to turn LPAgent data into a practical agent flow: discover pools, preview Zap-In actions, find skilled LP wallets, and know what to watch next.
```
